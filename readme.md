## Usage

```sh
# Show today's Sehar/Iftar.
roza
#npx roza

# Show help.
roza help
# Show all days.
npx roza beijing
npx roza karachi
npx roza lahore --all
npx roza lahore -a

# Show next day.
roza beijing -n
roza beijing --next

# For multiword cities.
npx roza "dera ghazi khan"
npx roza "dera ghazi khan" --all
npx roza "dera ghazi khan" -a
```
